In the project directory, you can run:

### `npm install` <br/>

or

### `pnpm instal` <br/>

Instal All dependencies in this project

### `npm run dev`

Runs the app in the development mode.<br />
Open [http://127.0.0.1:5173](http://127.0.0.1:5173) to view it in the browser.

### `npm run test`

Runs the app tests.
